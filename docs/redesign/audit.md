## C. Repository audit (condensed)

**Working (verified this session):** auth + sessions (Supabase), streaming chat w/ custom frames (meta/phase/token/mode/done), deterministic 4-level risk classifier + crisis templates + safety chips, journal (8 modes, autosave), mood check-ins (`logMood` → `["moods"]` invalidation), insights constellation (real tag math, SSR-deterministic), memories night sky (photo/video/audio), weekly letters, heal paths, urge shield, SOS, onboarding w/ consent, privacy page, settings/Sanctuary (memory controls, export, delete), dev-preview mode, 140/140 tests.

**Tech debt / limitations:** 2 known pre-existing type errors (route `server` option); screenshot-stalling animated backgrounds (heavy CSS animation — perf risk on mid-range Android, addressed by reduced-motion which exists); no audio assets (no audio exercises); insight evidence-linking (dates/excerpts per pattern) not built; helpfulness feedback ("did this help?") not persisted (feedback-loop design exists in docs/training).

**Security/privacy:** RLS owner-only on all tables; `is_ai_readable` consent gates; crisis numbers config-only; no E2E encryption → any E2E claim in references must NOT ship. Privacy page already uses honest wording.

**Features in references but missing:** stepped check-in journey (→ building now), Today hierarchy (→ now), Tools/You nav identity (→ now), insight-detail screen (deferred: needs evidence linking), audio player (deferred: no assets), helpfulness feedback loop (deferred: spec exists), pattern node-graph (deferred: honesty threshold).

## D. Reference inconsistencies resolved

1. Nav: 4 tabs vs 5 tabs vs 6 (Check-in tab) → **5 tabs: Today · InnerMate · Tools · Memories · You** (check-in reached from Today, not a tab).
2. Accent: violet-primary throughout references vs app's dawn-gold → **dual-accent**: violet = companion/AI presence + primary CTA; dawn-gold retained for warmth (stars, letters). Both exist in references (amber = confidence).
3. "Online" status → replaced by existing facet line ("listening, quietly").
4. Growth percentages → replaced by real counts (brief §15 mandates this).
5. "End-to-end encrypted" / "anonymous" → excluded (unverified; brief §Truth).
6. Streaks ("2 days") → gentle totals only, no streak-guilt.
